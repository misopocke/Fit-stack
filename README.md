# 筋トレ記録アプリ

JavaScriptベースの筋トレ記録アプリケーションです。ブラウザ上で動作し、トレーニングの記録、管理、進捗の可視化ができます。Apple Materialsデザインに基づいたシンプルでモダンなモノトーンデザインを採用しています。

## 概要

このアプリは、筋力トレーニングの記録を効率的に管理するためのツールです。階層的な種目選択、トレーニングセット管理、進捗グラフ表示、直前のセットをコピーする機能などを提供します。

## 機能

### 1. トレーニング記録
- **日付選択**: トレーニングを行った日付を選択
- **トレーニングセットから選択**: 日付の下にセット選択プルダウンを配置。セット管理で登録したセットを選択すると、そのセットに含まれるすべての種目がリスト表示され、順番に記録できます（任意）
- **種目リスト表示**: セット選択時に、セット内の種目リストが表示されます。現在記録中の種目はハイライト表示され、完了した種目は「完了」と表示されます
- **種目ナビゲーション**: 「前の種目」「次の種目」ボタンで種目を切り替えられます。記録を保存すると、自動的に次の種目に進みます
- **階層的種目選択**: 部位 → 器具タイプ → 種目の3段階で種目を選択（セット選択時は自動入力）
  - 部位: 胸、背中、脚、肩、腕、腹筋
  - 器具タイプ: マシン、フリーウェイト、自重、ケーブル
- **セット情報入力**: レップ数、重量、メモを入力
- **セットを複製**: 現在入力中の最終行のセット情報を複製
- **記録の保存・編集・削除**: 過去の記録を管理

### 2. トレーニングセット管理
- **セット作成**: カスタム名でトレーニングセットを作成
- **種目追加**: セットに複数の種目を追加
- **テンプレート機能**: 作成したセットをテンプレートとして使用
- **セットの編集・削除**: 保存したセットを管理

### 3. 記録履歴
- **履歴表示**: 過去の記録を一覧表示
- **日付フィルター**: 特定の日付の記録を表示
- **詳細情報**: 種目、セット数、重量、レップ数を表示

### 4. 進捗グラフ
- **種目別グラフ**: 選択した種目の進捗を可視化
- **重量推移**: 最大重量の推移を表示
- **レップ数推移**: 最大レップ数の推移を表示
- **Chart.js使用**: モノトーンデザインに合わせた美しいグラフ表示

## 使い方

### 基本的な使い方

1. `index.html`をブラウザで開きます
2. 「記録」タブでトレーニングを記録します
   - 日付を選択
   - （任意）「トレーニングセットから選択」でセットを選択すると、セット内の種目リストが表示されます
   - 種目リストで現在記録中の種目がハイライト表示されます
   - 部位 → 器具タイプ → 種目は自動的に選択されます（手動でも選択可能）
   - セット情報（レップ数、重量）を入力
   - 「記録を保存」をクリックすると、自動的に次の種目に進みます
   - 「前の種目」「次の種目」ボタンで種目を切り替えられます

### セットを複製

1. セット情報を入力します
2. 「セットを複製」ボタンをクリックすると、入力されている最終行のセット情報がその下に複製されます
3. 複製されたセット情報を必要に応じて修正できます

### トレーニングセットの作成

1. 「セット管理」タブを開きます
2. セット名を入力（例: 「上半身デー」）
3. 種目を追加（部位 → 器具タイプ → 種目を選択）
4. 「セットを保存」をクリック

### セットをテンプレートとして使用

1. 「記録」タブで「トレーニングセットから選択」プルダウンからセットを選択
2. セット内の種目リストが表示され、最初の種目が自動的に選択されます
3. セット情報を入力して「記録を保存」をクリック
4. 保存すると自動的に次の種目に進みます
5. 「前の種目」「次の種目」ボタンで種目を切り替えられます
6. 種目リストで完了した種目は「完了」と表示されます

### 進捗の確認

1. 「進捗グラフ」タブを開きます
2. 部位 → 器具タイプ → 種目を選択
3. グラフで重量とレップ数の推移を確認

## 技術スタック

- **HTML5**: 構造とマークアップ
- **CSS3**: Apple Materialsデザインに基づいたモノトーンデザイン、レスポンシブ対応
- **Vanilla JavaScript**: アプリケーションロジック
- **Chart.js**: 進捗グラフの表示（CDN経由）
- **localStorage**: データの永続化

## デザイン

このアプリは、Apple Materialsデザインガイドラインに基づいたデザインを採用しています：

- **モノトーンカラーパレット**: 白、ライトグレー、ダークグレー、ミディアムグレー
- **シンプルでクリーン**: 不要な装飾を排除し、機能性を重視
- **Appleシステムフォント**: 読みやすさと一貫性を確保
- **控えめな角丸**: 8px〜12pxの角丸でモダンな印象
- **細いボーダー**: 1pxのボーダーで要素を区切り
- **タブナビゲーション**: 下線でアクティブ状態を表示

## データ保存

データはブラウザのlocalStorageに保存されます。ブラウザの設定やキャッシュをクリアすると、データが削除される可能性があります。

## 種目データ

アプリには以下の部位と種目が登録されています：

- **胸**: チェストプレス、ベンチプレス、腕立て伏せなど
- **背中**: ラットプルダウン、デッドリフト、懸垂など
- **脚**: スクワット、レッグプレス、ランジなど
- **肩**: ショルダープレス、サイドレイズなど
- **腕**: アームカール、トライセップスなど
- **腹筋**: クランチ、プランク、レッグレイズなど

各部位には、マシン、フリーウェイト、自重、ケーブルの4つの器具タイプがあります。
